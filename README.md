## 💡 Getting Started

1. Clone the repo  
   `git clone https://github.com/yourusername/dans-xm-tech.git`

2. Install dependencies  
   `npm install`

3. Configure your `.env` file with API keys and credentials

4. Run the bot  
   `npm start`
